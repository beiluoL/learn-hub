这份路线把「Python 与数学基础」拆成 7 个阶段，每个阶段给出**核心任务**与**练手目标**。建议边看边敲，每完成一项就勾掉一项。

## 0. Python 数据科学速览

复习列表、字典、推导式与函数式工具

- [ ] 能写出列表推导
- [ ] 理解深浅拷贝

## 1. NumPy 基础

ndarray 创建、切片与广播机制

- [ ] 能 reshape 与索引
- [ ] 理解广播

## 2. 线性代数

矩阵乘法、转置、逆与特征分解

- [ ] 会做矩阵乘法
- [ ] 知道特征值含义

## 3. 微积分与梯度

导数、偏导数与梯度下降直觉

- [ ] 理解梯度下降更新公式
- [ ] 调过学习率

## 4. 概率与统计

分布、期望、方差与贝叶斯

- [ ] 会采样与算统计量
- [ ] 理解贝叶斯

## 5. Pandas 数据处理

DataFrame 选取、分组与缺失值

- [ ] 会 groupby
- [ ] 能处理 NaN

## 6. 向量化与性能

用向量化替代循环提速

- [ ] 写出向量化代码
- [ ] 对比循环性能

## 资源与节奏

- 官方文档与权威资料优先；
- 节奏：每天 1~2 小时，理论 + 敲代码交替；
- 关键：每个模块至少完成 1 个可运行的小项目，代码放仓库。
