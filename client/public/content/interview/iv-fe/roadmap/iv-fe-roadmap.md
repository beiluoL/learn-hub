这份路线把「前端面试」拆成 7 个阶段，每个阶段给出**核心任务**与**练手目标**。建议边看边敲，每完成一项就勾掉一项。

## 0. JS 闭包与作用域

词法作用域、闭包形成与经典坑

- [ ] 能解释闭包原理
- [ ] 能举出应用场景

## 1. 事件循环与宏微任务

Event Loop、Promise 与渲染时机

- [ ] 能预测输出顺序
- [ ] 能说宏微任务区别

## 2. 原型与继承

原型链、class 与寄生组合继承

- [ ] 能画原型链
- [ ] 能说继承实现

## 3. 浏览器渲染与重排重绘

关键渲染路径、回流与重绘优化

- [ ] 能说渲染流程
- [ ] 能列优化手段

## 4. HTTP 与缓存

状态码、强缓存与协商缓存

- [ ] 能区分两类缓存
- [ ] 能说状态码含义

## 5. Vue 响应式原理

Object.defineProperty 与 Proxy 响应式

- [ ] 能说响应式流程
- [ ] 能对比 2 与 3

## 6. 前端工程化

构建工具、模块化与 CI

- [ ] 能说 ESM 与 CJS 区别
- [ ] 能说 Tree Shaking

## 资源与节奏

- 官方文档与权威资料优先；
- 节奏：每天 1~2 小时，理论 + 敲代码交替；
- 关键：每个模块至少完成 1 个可运行的小项目，代码放仓库。
