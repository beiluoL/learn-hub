这份路线把「Java 基础语法」拆成 6 个阶段，每个阶段给出**核心任务**与**练手目标**。建议边看边敲，每完成一项就勾掉一项。

## 0. 环境与第一个程序

搭建 JDK 环境，理解编译运行流程，写出第一个 Java 程序。

- [ ] 装好 JDK 并配置环境变量
- [ ] 能独立编译运行 Hello World
- [ ] 理解包与目录的对应关系

## 1. 变量与数据类型

掌握 8 种基本类型、引用类型与类型转换。

- [ ] 能区分基本类型与引用类型
- [ ] 熟练使用强制/自动转换
- [ ] 理解 final 语义

## 2. 运算符与流程控制

算术/关系/逻辑运算符，if/switch 与三种循环。

- [ ] 能用三种循环实现遍历
- [ ] 理解短路逻辑
- [ ] 掌握 switch 新语法

## 3. 数组与字符串

一维/多维数组，String 不可变性与常用 API。

- [ ] 理解数组与多维数组
- [ ] 说清 String 不可变原因
- [ ] 会用 StringBuilder

## 4. 方法与方法重载

方法定义、参数传递、重载与递归。

- [ ] 理解值传递本质
- [ ] 能写出方法重载
- [ ] 写过递归（如阶乘）

## 5. 常用类与工具

Math、日期时间、Scanner、包装类等日常工具。

- [ ] 会用 java.time 处理日期
- [ ] 掌握 Scanner 与格式化
- [ ] 理解包装类缓存

## 资源与节奏

- 官方文档与权威资料优先；
- 节奏：每天 1~2 小时，理论 + 敲代码交替；
- 关键：每个模块至少完成 1 个可运行的小项目，代码放仓库。
