- `Collection`：List（有序可重复）、Set（不可重复）、Queue（队列）。
- `Map`：键值对，key 不可重复。
- 泛型 `<T>` 提供编译期类型安全。
- 统一迭代方式：`for-each` 与 `Iterator`。
- 选型的本质是时间与空间复杂度的权衡。
