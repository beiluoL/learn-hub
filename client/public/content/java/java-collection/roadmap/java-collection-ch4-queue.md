- `Queue`：FIFO，`offer/poll/peek`。
- `Deque` 可作双端队列或栈（`push/pop`）。
- `PriorityQueue`：堆实现，按优先级出队。
- `ArrayDeque` 性能优于 `Stack`。
- 阻塞队列用于并发（见并发模块）。
