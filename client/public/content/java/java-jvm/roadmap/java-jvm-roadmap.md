这份路线把「JVM 与调优」拆成 6 个阶段，每个阶段给出**核心任务**与**练手目标**。建议边看边敲，每完成一项就勾掉一项。

## 0. 内存结构

运行时数据区：堆、栈、方法区与直接内存。

- [ ] 说清各区域职责
- [ ] 理解堆与栈区别
- [ ] 知道元空间

## 1. 垃圾回收算法

标记清除/复制/整理与分代收集思想。

- [ ] 理解可达性分析
- [ ] 区分三种算法
- [ ] 理解分代思路

## 2. 垃圾收集器

Serial/Parallel/CMS/G1/ZGC 演进。

- [ ] 能对比主流收集器
- [ ] 知道 G1 特点
- [ ] 了解 ZGC

## 3. 类加载机制

双亲委派与自定义类加载器。

- [ ] 说清加载流程
- [ ] 理解双亲委派
- [ ] 能自定义加载器

## 4. 运行时参数与工具

常用 JVM 参数与诊断命令。

- [ ] 会设堆参数
- [ ] 会用 jps/jstat
- [ ] 会用 jstack

## 5. 调优实战

OOM 与 CPU 飙高的定位思路。

- [ ] 能定位 OOM
- [ ] 会查 CPU 飙高
- [ ] 有调优闭环

## 资源与节奏

- 官方文档与权威资料优先；
- 节奏：每天 1~2 小时，理论 + 敲代码交替；
- 关键：每个模块至少完成 1 个可运行的小项目，代码放仓库。
