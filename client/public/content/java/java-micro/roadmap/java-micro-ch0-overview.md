- 单体 → 微服务：按业务域拆分独立部署。
- 核心组件：注册中心、配置中心、网关、链路追踪。
- 优势：独立扩展、技术异构、故障隔离。
- 代价：分布式复杂度、运维、网络开销、一致性。
- 不是银弹，小团队慎用。
