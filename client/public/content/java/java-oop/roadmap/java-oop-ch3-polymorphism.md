- 父类引用指向子类对象：`Animal a = new Cat();`。
- 运行时根据实际类型调用重写方法（动态绑定）。
- 多态解耦调用方与具体实现。
- 抽象类 `abstract` 不能实例化，强制子类实现。
- 面向接口编程是多态的高级形态。
