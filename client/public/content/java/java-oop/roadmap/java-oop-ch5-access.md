- `package com.xxx;` 必须位于对应目录。
- `import` 引入类，通配符 `import com.xxx.*`。
- 访问控制：private/默认/protected/public 的可见范围。
- `static import` 可直接引入静态成员。
- 用 `jar` 打成可复用组件。
