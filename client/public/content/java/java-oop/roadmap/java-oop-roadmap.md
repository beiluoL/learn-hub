这份路线把「面向对象」拆成 6 个阶段，每个阶段给出**核心任务**与**练手目标**。建议边看边敲，每完成一项就勾掉一项。

## 0. 类与对象

理解类作为模板、对象作为实例，掌握构造器与 this。

- [ ] 能定义类与构造器
- [ ] 理解 this 作用
- [ ] 区分成员与局部变量

## 1. 封装

用 private 隐藏细节，通过 getter/setter 暴露可控访问。

- [ ] 能给字段加 private
- [ ] 会写 getter/setter
- [ ] 理解封装收益

## 2. 继承

extends 复用父类，super 调用父类，理解重写与 Object。

- [ ] 会用 extends
- [ ] 理解 super 调用
- [ ] 正确重写方法

## 3. 多态

向上转型与动态绑定，接口/抽象类实现多态。

- [ ] 理解向上转型
- [ ] 能解释动态绑定
- [ ] 用过抽象类

## 4. 接口与抽象类

interface 定义契约，支持默认方法与多实现。

- [ ] 能定义并实现接口
- [ ] 理解函数式接口
- [ ] 区分接口与抽象类

## 5. 包与访问控制

用 package 组织代码，理解访问修饰符与 jar 打包。

- [ ] 会用 package 组织代码
- [ ] 理解四种访问修饰符
- [ ] 能打 jar 包

## 资源与节奏

- 官方文档与权威资料优先；
- 节奏：每天 1~2 小时，理论 + 敲代码交替；
- 关键：每个模块至少完成 1 个可运行的小项目，代码放仓库。
