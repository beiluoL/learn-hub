这份路线把「Spring / Boot」拆成 6 个阶段，每个阶段给出**核心任务**与**练手目标**。建议边看边敲，每完成一项就勾掉一项。

## 0. IoC 与 DI

控制反转、依赖注入与 Bean 定义。

- [ ] 理解 IoC/DI
- [ ] 会用注解注册 Bean
- [ ] 用构造器注入

## 1. Bean 生命周期

作用域、后置处理器与循环依赖。

- [ ] 理解作用域
- [ ] 理解生命周期
- [ ] 知道循环依赖处理

## 2. AOP 面向切面

切面、通知与事务注解原理。

- [ ] 理解 AOP 概念
- [ ] 能写切面
- [ ] 理解事务注解原理

## 3. Spring Boot 自动配置

@SpringBootApplication 与 starter 机制。

- [ ] 理解自动配置
- [ ] 会用 starter
- [ ] 会用条件注解

## 4. Web 开发

REST 接口、参数绑定与统一处理。

- [ ] 能写 REST 接口
- [ ] 会参数绑定
- [ ] 统一异常处理

## 5. 数据访问集成

Spring Data JPA 与 MyBatis 整合。

- [ ] 会用 JPA/MyBatis
- [ ] 理解事务管理
- [ ] 能整合数据源

## 资源与节奏

- 官方文档与权威资料优先；
- 节奏：每天 1~2 小时，理论 + 敲代码交替；
- 关键：每个模块至少完成 1 个可运行的小项目，代码放仓库。
