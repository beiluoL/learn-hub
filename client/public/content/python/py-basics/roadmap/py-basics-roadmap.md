这份路线把「Python 基础语法」拆成 7 个阶段，每个阶段给出**核心任务**与**练手目标**。建议边看边敲，每完成一项就勾掉一项。

## 0. 环境与第一个程序

搭建 Python 运行环境并运行第一段代码

- [ ] 能打印出 hello
- [ ] 成功创建并激活虚拟环境
- [ ] 知道如何退出虚拟环境

## 1. 变量与基本类型

理解变量、数字、字符串与布尔类型

- [ ] 能区分 int 与 float
- [ ] 会使用 f-string 格式化输出
- [ ] 理解 None 的含义

## 2. 流程控制

if/for/while 控制程序走向

- [ ] 能用 for 输出 0-4
- [ ] 理解 break 与 continue 区别
- [ ] 代码缩进一致无报错

## 3. 字符串处理

切片、格式化与常用方法

- [ ] 能把逗号字符串拆成列表
- [ ] 会反转字符串
- [ ] 掌握 strip 去空格

## 4. 容器类型

列表、元组、字典与集合

- [ ] 会写列表推导式
- [ ] 理解 list 与 tuple 区别
- [ ] 用 set 去重

## 5. 函数定义

参数、返回值与默认参数

- [ ] 能定义带默认参数的函数
- [ ] 会用 *args 接收变长参数
- [ ] 写出可读 docstring

## 6. 文件读写

with 上下文管理器安全读写文件

- [ ] 用 with 读写文件不漏关
- [ ] 能按行读取文本
- [ ] 理解 utf-8 编码

## 资源与节奏

- 官方文档与权威资料优先；
- 节奏：每天 1~2 小时，理论 + 敲代码交替；
- 关键：每个模块至少完成 1 个可运行的小项目，代码放仓库。
